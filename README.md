### Siber Userbot Otomatik Deployer

[Siri UserBot Reposu](https://github.com/yusiqo/SiberUserBot)

## Kurulum
```sh
git clone https://github.com/yusiqo/siberuserbot 
cd installer
pip install -r requirements.txt
python3 -m siri_installer
```
